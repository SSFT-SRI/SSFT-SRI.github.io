Consider the theory [[file:UTM0.pvs][~UTM0~]] and the lecture [[file:../lecture-1.pdf][Developing and Proving Algorithms with PVS (Part I)]].

1. Using ~remove_op~, define the function [[file:UTM0.pvs#L55][~unsubscribe~]] that unsubscribes any operation of aircraft ~ac_id~ from ~ops~.
2. Define a type judgement [[file:UTM0.pvs#L67][~unsubscribe_j~]] for the function ~unsubscribe~ stating that ~result~, the output of the function, does not have an operation for aircraft ~ac_id~, the input parameter of the function. Prove the type judgement using ~remove_op_j~ as a lemma.
3. Fix [[file:UTM0.pvs#L110][~test_1~]] so that it holds. *Hint*: What is known about ~Ac0~, ~Ac1~, and ~Ac2~? What should be assumed?
4. Fix [[file:UTM0.pvs#L32][~subscribe~]] so that only one operation per aircraft can be subscribed. After the change, make sure all TCCs and LEMMAS are still valid.
5. Define the function [[file:UTM0.pvs#L129][~update~]] that modifies the operation for a given aircraft. The function does nothing if the aircraft is not subscribed. Write a lemma to test the functionality. *Hint*: First, define a recursive function on ~Operations~ that changes an element in the list. Then, use this function to define ~update~.
